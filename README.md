# Selene Wallet

## Live Demo
- https://app.selene.cash/

## Developer Quick Start
1. `git clone https://git.xulu.tech/selene.cash/selene-wallet`
2. `cd selene-wallet`
3. `yarn install`
4. `yarn run build`
5. `yarn dev`

## Building for Android
1. Install Android Studio
2. `npx cap sync`
3. `npx cap run android`
